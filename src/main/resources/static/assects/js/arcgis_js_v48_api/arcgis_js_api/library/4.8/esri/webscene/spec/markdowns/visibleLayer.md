# visibleLayer

Object with layer id, used to indicate layer visibility in a slide.

### Properties

| Property | Details
| --- | ---
| id | The id of the layer as listed on the operational layer.


### Example

```json
{
  "visibleLayers": [
    {
      "id": "73128eb3-05ee-46a7-8c22-0c385a0b1827"
    }
  ]
}
```
