# sizeStop

A `sizeStop` object describes the size of the symbol at various values of the expression.

### Properties

| Property | Details
| --- | ---
| size | Specifies the marker size to use for the specified value.
| value | The value to be mapped to a size.


### Example

```json
{
  "stop": {
    "value": 0,
    "size": 255
  }
}
```
